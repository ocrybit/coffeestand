Class Properties



@root (String) : the root path to start walking 

@ignoreFiles (Array) : glob Strings to match files and directories to ignore 

@dirs (Array) : stat Objects of found directories 

@files (Array) : stat Objects of found files 

@readCount (Number) : a count to keep track of async readdir processes 

@callbacks (Object) : callback Functions to execute when files and directories are found,

Check if a directory should be ignored



dir (String) : a directory path to check  
             		                            
  _isIgnore: (dir) ->
    for v in @ignoreFiles
      if minimatch(dir, v)
        return true
    return false
